My CapsLock Arrows Instructions

[功能:按住修饰键[CapsLock（其实被映射成了Fn键）时]
	1: i/j/k/l => up/left/down/right.
	2: u/o => 行内字符串首/尾.(command+left/right)
	2: n和/ => 行首/行尾.(control+a/e)(不同于home/end)
	3: y/h => pageUp/pageDown.
	4: 7/8 => 选中直至上一个/下一个字符串首/尾
	5: 9/0 => 将光标移至上一个/下一个字符串首/尾
	6: m/,/. => backspace/delete/enter.
	7: q => Escape.
	8: spacebar => left_control+spacebar.(切换输入法)
	9: enter => capslock.(还原物理键CapsLock键的原生功能)
